Tipos de valor:
 - Numéricos: sbyte, short, int, long, byte, ushort, uint, ulong
 - Caracteres Unicode: char
 - Pontos floar: float, double, decimal
 - Booleano: bool
 - enun, struct e tipos nullable

Tipos de referência:
 - Tipos classe: class, object, string
 - Tipos arrays: int[], int[,], etc...
 - interface, delegate

Instruções:
 - Declaração de variáveis e constantes locais
 - if
 - else
 - switch
 - while
 - do
 - for 
 - foreach
 - break
 - continue 
 - return
 - throw 
 - try .. catch .. finally
 - using

Arrays:
 - Basicamente o vetor da linguagem C
 - Tem tamanho fixo e especificado
 - Os índices dos elementos vão de 0 até o comprimento do array-1
 - Pode ter várias dimensões
